# 🎵 Background Music

Place background music files (.mp3) here, organized by category:

```
data/music/
├── energetica/       → Upbeat, motivational tracks
│   ├── track_01.mp3
│   └── track_02.mp3
├── calma/            → Soft, ambient tracks
│   └── ambient_01.mp3
├── urgente/          → High-energy, urgency-inducing tracks
│   └── urgency_01.mp3
└── neutra/           → Neutral, corporate-style tracks
    └── neutral_01.mp3
```

## Requirements
- **Format:** MP3 (128kbps+)
- **Duration:** At least 60 seconds (loops automatically)
- **Licensing:** Use royalty-free music only
- **Volume:** Will be mixed at -30dB relative to narration

## Sources (royalty-free)
- [Pixabay Music](https://pixabay.com/music/)
- [Mixkit](https://mixkit.co/free-stock-music/)
- [Free Music Archive](https://freemusicarchive.org/)
